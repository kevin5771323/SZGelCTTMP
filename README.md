# 家庭大厨微信小程序+SSM项目

## 前言

在这个快节奏的生活中，烹饪成为了一种享受生活的体现。家庭大厨微信小程序+SSM项目旨在为广大用户提供便捷、高效的烹饪体验。本项目结合了Java技术、微信小程序以及前端技术，致力于打造一款实用性强的家庭烹饪助手。

## 内容介绍

家庭大厨微信小程序+SSM项目是一款集菜谱推荐、食材选购、烹饪技巧分享等功能于一体的微信小程序。用户可以通过该项目轻松浏览各类菜谱，根据个人口味进行食材选购，同时还可以学习到实用的烹饪技巧。此外，本项目还提供了在线交流平台，让用户能够互相分享烹饪心得，共同进步。

## 技术介绍

- 语言：Java
- 使用框架：Spring、Spring MVC、MyBatis
- 微信小程序：微信小程序
- 前端技术：JS、Vue、CSS3、Uniapp
- 开发工具：IDEA/Eclipse、Uniapp
- 数据库：MySQL 5.7/8.0
- 数据库管理工具：phpstudy/Navicat
- JDK版本：jdk1.8
- Maven：apache-maven 3.8.1-bin
- 前端环境：Node.Js 12\14\16
